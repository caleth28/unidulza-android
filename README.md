# Unidulza

App Android nativa para Universal enfocada en distribuidores de postres y reposteria.

## Concepto

Unidulza es una experiencia B2B moderna inspirada en la agilidad de Diadia, pero aterrizada a productos Universal: gelatinas, flanes, mazamorras, coberturas, insumos y promociones para negocios de reposteria.

## Por que este nombre

Unidulza combina "Universal" + "dulzura". Es corto, memorable y directo para una marca de postres. En una busqueda web rapida no aparecio como app o marca de reposteria consolidada; antes de publicar conviene validar registro formal de marca.

## Flujo incluido

- Inicio con identidad visual moderna.
- Categorias de productos por ocasion.
- Productos con precios, descuento, stock y unidades por caja.
- Pantalla de promociones.
- Carrito funcional con cantidades, ahorro, minimo de pedido y total por caja.
- Confirmacion de pedido local.
- Seguimiento visual del ultimo pedido.
- Historial local de pedidos.
- Panel de negocio sin login.

## Abrir en Android Studio

1. Abre esta carpeta como proyecto Android Studio.
2. Espera la sincronizacion Gradle.
3. Ejecuta `app` en un emulador o equipo Android.

## APK generado

El APK debug queda en:

`app/build/outputs/apk/debug/app-debug.apk`

## Siguientes modulos recomendados

- Catalogo real conectado a API o Firebase.
- Carrito persistente.
- Reglas de precio por zona y volumen.
- Estados de despacho conectados al ERP.
- Imagenes reales de productos Universal.
- Login de cliente distribuidor cuando se defina el acceso por RUC/codigo.
